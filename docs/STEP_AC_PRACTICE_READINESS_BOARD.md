# STEP AC — Practice readiness board

## Obiettivo
Aggiungere nella tab Pratica un quadro di completezza operativa generale, oltre al solo blocco soggetti collegati.

## Cosa introduce
- board di completezza operativa
- lettura per blocchi: soggetti, nodi logistici, trasporto/tempi, documenti/dogana
- stato per blocco: pronto / da rifinire / priorità alta
- azione rapida "Vai al punto" verso il primo campo mancante del blocco
- overview generale con prossimo punto operativo
