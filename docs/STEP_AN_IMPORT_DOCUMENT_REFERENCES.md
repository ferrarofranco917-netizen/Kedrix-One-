# STEP AN — Import riferimenti documentali / metadata

## Obiettivo
Introdurre nel modulo Documenti un import controllato di riferimenti documentali collegati alle pratiche esistenti, senza richiedere ancora l'allegato binario.

## Cosa abilita
- pannello Import riferimenti documentali nel modulo Documenti
- mapping CSV/Excel
- validazione collegamento pratica
- commit controllato su metadata documentali
- visibilità dei riferimenti nel Document Engine come elementi `solo riferimento`

## Vincoli rispettati
- nessun refactor distruttivo della tab Pratica
- nessuna crescita monolitica dei file import esistenti
- traduzioni IT/EN/ES/FR allineate
- file binari allegati sempre separati dal metadata import
