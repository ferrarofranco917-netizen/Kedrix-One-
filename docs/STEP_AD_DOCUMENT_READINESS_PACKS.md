# STEP AD — Practice document readiness packs

## Obiettivo
Rafforzare la tab Pratica con una lettura documentale più vicina a documenti, tracking e fatturazione.

## Cosa aggiunge
- blocco **Completezza documentale**
- pacchetti documentali per tipo pratica
- confronto tra riferimenti inseriti e allegati presenti
- azioni rapide:
  - Vai ai riferimenti
  - Vai agli allegati

## Pacchetti letti
- dossier trasporto
- dogana / compliance
- commerciale / supporto
- flusso / riferimento (magazzino)

## File toccati
- index.html
- js/practices/practice-document-readiness.js
- js/practices/practice-overview.js
- js/app.js
- js/i18n.js
- style.css
- sw.js

## Test rapido
1. aprire una pratica con dati parziali
2. verificare il blocco Completezza documentale
3. usare Vai ai riferimenti
4. usare Vai agli allegati
5. importare almeno un allegato e verificare aggiornamento stato
6. controllare stabilità di Allegati, Dettaglio e Quadro soggetti
