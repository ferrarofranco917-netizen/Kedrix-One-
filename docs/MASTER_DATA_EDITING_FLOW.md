# Kedrix One — STEP R

## Anagrafiche come sottomoduli leggibili/modificabili

Questo step introduce una gestione reale del modulo Anagrafiche:

- elenco filtrabile per famiglia
- apertura di una scheda esistente
- modifica e salvataggio della scheda aperta
- creazione nuova scheda senza uscire dal modulo
- mantenimento del quick add verso la Pratica
- compatibilità con il lookup Partita IVA già introdotto

## Famiglie coperte

### Scheda completa strutturata
- Cliente
- Importatore
- Destinatario
- Mittente
- Vettore

### Directory apribili e modificabili
- Dogane
- Origini
- Destinazioni
- Codici articolo
- Compagnie marittime
- Compagnie aeree
- Tipologie unità/trasporto
- Navi
- TARIC

## Flusso operativo
1. Apri Anagrafiche
2. Scegli la famiglia
3. Cerca oppure apri una riga esistente
4. Modifica i dati nella scheda a destra
5. Salva modifiche
6. Se entri da quick add, il ritorno alla Pratica resta automatico
