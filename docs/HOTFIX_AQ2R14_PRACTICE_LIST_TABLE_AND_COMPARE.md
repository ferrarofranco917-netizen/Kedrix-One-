# AQ2R14 — Gestione pratiche: tabella più forte e confronto periodale

## Obiettivo
Rafforzare Gestione pratiche lato tabella senza toccare il workspace di lavoro della pratica.

## Contenuto dello step
- ordinamento reale dell’elenco pratiche
- selettore sort by + verso
- confronto per direzione (import / export / magazzino)
- tabella risultati con colonne più utili
- badge bozza incompleta / completa
- riferimenti trasporto raggruppati nella riga elenco
