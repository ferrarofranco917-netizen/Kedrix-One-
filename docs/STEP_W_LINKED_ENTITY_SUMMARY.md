# STEP W — Linked entity contextual summary

## Obiettivo
Rendere la tab Pratica più leggibile operativamente quando un soggetto collegato è già agganciato a un'anagrafica strutturata.

## Cosa aggiunge
- mini-summary contestuale sotto i campi relazionali collegati
- dati rapidi leggibili senza aprire subito l'anagrafica
- supporto su Cliente, Importatore, Destinatario, Mittente, Vettore e altri campi strutturati collegati

## Dati mostrati
- P.IVA o codice
- località
- contatto (email / telefono)
- indirizzo, se disponibile
